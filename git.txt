Basic commands

-git init: tạo ra 1 repository trong máy
-git clone: lấy trên mạng về
-git pull: đồng bộ trên mạng về máy

-git add vs git add: thêm file, chuẩn bị sẵn sàng để commit  
--> git add git.cpp
git add . -> add tat cac cac file
-git commit: sửa
--> git commit -m "Message to be sent"
-git push: đẩy lên đồng bộ máy vs mạng 

-git log : check status
-git log - decorate -graph -oneline
